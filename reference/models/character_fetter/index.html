    Character Fetter Models

    character_fetter

    Classes:

    • CharacterFetter –

      Container for character quotes and stories (fetter information).

    • Quest –

      Represents a quest related to unlocking a character quote or story.

    • Quote –

      Represents a character voice-over quote.

    • Story –

      Represents a character story entry.

    • Task –

      Represents a task or condition required to unlock a character quote.

    CharacterFetter pydantic-model

    Container for character quotes and stories (fetter information).

    Attributes:

    • quotes (list[Quote]) –

      A list of character voice-over quotes.

    • stories (list[Story]) –

      A list of character story entries.

    Fields:

    • quotes (list[Quote])
    • stories (list[Story])

    Validators:

    • _flatten_quotes → quotes
    • _flatten_stories → stories

    quotes pydantic-field 

    quotes: list[Quote]

    stories pydantic-field 

    stories: list[Story]

    Quest pydantic-model

    Represents a quest related to unlocking a character quote or story.

    Attributes:

    • id (int) –

      The ID of the quest.

    • quest_title (str | None) –

      The title of the specific quest (optional).

    • chapter_id (int) –

      The ID of the chapter the quest belongs to.

    • chapter_title (str) –

      The title of the chapter the quest belongs to.

    Fields:

    • id (int)
    • quest_title (str | None)
    • chapter_id (int)
    • chapter_title (str)

    chapter_id pydantic-field 

    chapter_id: int

    chapter_title pydantic-field 

    chapter_title: str

    id pydantic-field 

    id: int

    quest_title pydantic-field 

    quest_title: str | None = None

    Quote pydantic-model

    Represents a character voice-over quote.

    Attributes:

    • title (str) –

      The title or category of the quote (e.g., "Chat: Knights").

    • audio_id (str) –

      The identifier for the audio file associated with the quote.

    • text (str) –

      The transcribed text of the quote.

    • tips (str | None) –

      Optional tips or context related to the quote.

    • tasks (list[Task]) –

      A list of tasks required to unlock this quote.

    Fields:

    • title (str)
    • audio_id (str)
    • text (str)
    • tips (str | None)
    • tasks (list[Task])

    Validators:

    • _format_text → text
    • _convert_empty_tips → tips
    • _convert_empty_tasks → tasks

    audio_id pydantic-field 

    audio_id: str

    tasks pydantic-field 

    tasks: list[Task]

    text pydantic-field 

    text: str

    tips pydantic-field 

    tips: str | None

    title pydantic-field 

    title: str

    Story pydantic-model

    Represents a character story entry.

    Attributes:

    • title (str) –

      The title of the story entry (e.g., "Character Story 1").

    • title2 (str | None) –

      An alternative or secondary title (optional).

    • text (str) –

      The main text content of the story entry.

    • text2 (str | None) –

      Alternative or secondary text content (optional).

    • tips (str | None) –

      Optional tips or context related to the story entry.

    Fields:

    • title (str)
    • title2 (str | None)
    • text (str)
    • text2 (str | None)
    • tips (str | None)

    Validators:

    • _format_text → text
    • _format_text2 → text2
    • _convert_empty_tips → tips

    text pydantic-field 

    text: str

    text2 pydantic-field 

    text2: str | None

    tips pydantic-field 

    tips: str | None

    title pydantic-field 

    title: str

    title2 pydantic-field 

    title2: str | None

    Task pydantic-model

    Represents a task or condition required to unlock a character quote.

    Attributes:

    • type (str) –

      The type of task (e.g., "FETTER_TASK_FINISH_QUEST").

    • quest_list (list[Quest]) –

      A list of quests associated with this task.

    Fields:

    • type (str)
    • quest_list (list[Quest])

    quest_list pydantic-field 

    quest_list: list[Quest]

    type pydantic-field 

    type: str
